stickyBroadcast

  • stickyBroadcast<keyType>(key: keyType, value?: ValueType[broadcast.ToAnyIndexKey<keyType, ValueType>], callback?: ResultCallBack<ResultType[broadcast.ToAnyIndexKey<keyType, ResultType>]>, persistence?: boolean): void
    • Defined in broadcast/src/broadcast.ts:221

    广播一条 指定 [key] 的粘性消息 如果广播系统中没有注册该类型的接收者，本条信息将被滞留在系统中。一旦有该类型接收者被注册，本条消息将会被立即发送给接收者 如果系统中已经注册有该类型的接收者，本条消息将会被立即发送给接收者。

    Type parameters

    • keyType: keyof MsgKeyType = any

    Parameters

    • key: keyType

      消息类型

    • Optional value: ValueType[broadcast.ToAnyIndexKey<keyType, ValueType>]

      消息携带的数据。可以是任意类型或是null

    • Optional callback: ResultCallBack<ResultType[broadcast.ToAnyIndexKey<keyType, ResultType>]>

      能够收到接收器返回的消息

    • Optional persistence: boolean

      是否持久化消息类型。持久化的消息可以在任意时刻通过 broadcast.value(key) 获取当前消息的数据包。默认情况下，未持久化的消息类型在没有接收者的时候会被移除，而持久化的消息类型则不会。开发者可以通过 [clear] 函数来移除持久化的消息类型。

    Returns void
